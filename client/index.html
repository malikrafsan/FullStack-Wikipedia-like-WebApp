WIKI-ARC NETWORKING

Wiki-ARC merupakan wikipedia-like web app yang dibuat oleh Malik Akbar Hashemi Rafsanjani (16520299) guna memenuhi tugas Day 5 ARC. Wiki-ARC menyediakan tempat untuk menuliskan tulisan tentang Networking, dan dapat diedit oleh siapa saja yang mengakses, selayaknya wikipedia. Web app ini menggunakan mysql sebagai database dan menyediakan fitur CRUD dalam menjalankan fungsinya.
